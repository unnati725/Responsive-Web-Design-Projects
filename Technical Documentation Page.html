Introduction

Python is a popular programming language.

It is easy to learn and widely used.

print("Hello World")
  • Easy syntax
  • Open source
Variables

Variables store data values.

You can assign values using the equals sign.

x = 10
  • Store numbers
  • Store text
Data Types

Python has multiple built-in data types.

Examples include strings and integers.

name = "Unnati"
  • String
  • Integer
Functions

Functions help organize code.

They can accept parameters and return values.

def greet(): print("Hello")
  • Reusable code
  • Improves readability
Loops

Loops repeat a block of code.

Python supports for and while loops.

for i in range(5): print(i)
  • for loop
  • while loop
